# Gap Map — Master Parts Bin vs. Live Site

Read-only analysis. Compares the founder's approved "Master Parts Bin" (voice/copy buckets, Parts 1–4) against the current live site copy (`docs/_work/site-copy-inventory.md`, sourced from `artifacts/wlc-site/src/pages/*.tsx`).

Each section gives three lists:
- **On site AND in buckets** — already aligned.
- **In buckets, NOT yet on site** — strong approved lines/sections we haven't used (the opportunity).
- **On site, NOT in buckets / off-voice** — candidates to cut or rewrite. Flagged: generic phrasing, or banned words (continuity, stewardship, curated).

Banned-word note: the bin itself uses "continuity" heavily (House Calls, Legacy). Where the live site uses it, it's flagged as a banned-word risk regardless of bin precedent.
